Project

苏职大智能制造博物馆

时间：2020.04.01 - 2021.02.03

应用类型：web，h5，webgl，pc，mobile

项目概述：为 苏州职业大学 定制的一个webgl版本的全景博物馆展厅。此web应用主要由：前台全景网站 web，后台管理系统 web，后台管理系统 server ，各种资源页面（h5，video、webgl） 四个部分构成。

功能概述：

  1. 全景平台 web：预加载页；3d全景模块（可以通过鼠标、键盘 放大，缩小，旋转3d场景；场景切换热点，跳转相邻的场景；墙壁资源热点，调取对应的资源页面；动态热点，带文字提示；场景切换时loading效果）；工具栏模块（帮助：开启、关闭 操作帮助页；全屏：开启，关闭 全屏；3d场景操作模式：拖拽旋转3d场景，滑屏自动旋转3d场景；背景音乐；是否自动播放3d场景旋转动画；当前场景资源页缩列图列表：点击缩列图，跳转到对应的资源页面；网站统计：总浏览量，今日浏览量，资源浏览量Top10列表；刷新当前3d场景模块；场景地图模块：导航地图，可以切换导航点，跳转到不同的场景；当前对应场景，热点加亮显示；）；数据统计模块（统计：总浏览量、今日浏览量、资源页浏览量、资源页访问时长；页面显示：总、今日浏览量；）；移动端优化;
  2. 资源页模块：包含：各种h5动画，h5测试题，webgl页面（n种模式，全用的threejs写的） 总共400多个资源页；移动端优化;
  3. 后台管理系统 web：登录；数据统计：浏览量模块（总浏览量；今日浏览量；所有资源页面的浏览量，访问总时间列表；表格：下载，密度，刷新，列设置；）；地域分布模块（浏览量：按省 地图显示；按国家 地图显示；数据视图模块，下载为图片功能，重置地图设置功能；地图可以拖动，放大缩小；）
  4. 服务端 server：各种接口，以及相应的功能实现；数据统计；https安全认证；

本人职责： 负责 前台全景网站 web，资源页模块，后台管理系统 web，后台管理系统 server 全部功能开发。

技术参数：

  1. 前台全景网站 web：react（cra），mobx，hook，threejs，createjs，antd，axios，css in module等；
  2. 资源页模块：react，threejs，createjs，antd，axios，hook 等；
  3. 后台管理系统 web：antd pro ，echarts；
  4. 后台管理系统 server：eggjs（nodejs框架），lowdb（微型数据库），pkg, nginx；

软件工具：WebStorm，3dsmax，Blender，Adobe Animate CC
