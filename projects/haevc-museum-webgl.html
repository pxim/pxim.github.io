Project

湖南汽车工程职业学院 汽车博物馆 WebGL

时间：2017.04.19 - 2017.11

应用类型：web，h5，webgl，pc，mobile

项目概述：为 湖南汽车工程职业学院 定制的一个webgl版本的全景博物馆展厅。此web应用主要由：前台博物馆网站 webgl，后台管理系统 web，后台管理系统 server ，各种资源页面（h5，video、webgl） 四个部分构成。

功能概述：

  1. 前台博物馆网站 webgl：预加载页；3d全景模块（可以通过鼠标、键盘 放大，缩小，旋转3d场景；场景切换热点，跳转相邻的场景；墙壁资源热点，调取对应的资源页面；动态热点，带文字提示；场景切换时loading效果）；工具栏模块（帮助：开启、关闭 操作帮助页；全屏：开启，关闭 全屏；3d场景操作模式：拖拽旋转3d场景，滑屏自动旋转3d场景；背景音乐；是否自动播放3d场景旋转动画；当前场景资源页缩列图列表：点击缩列图，跳转到对应的资源页面；刷新当前3d场景模块；场景地图模块：导航地图，可以切换导航点，跳转到不同的场景；当前对应场景，热点加亮显示；）；

  2. 资源页模块：包含：各种h5动画，h5测试题，webgl页面 总共300多个资源页；

  3. 后台管理系统 web：登录；背景音乐管理（上传，更新）；配置表管理（下载，上传，更新）；

  4. 后台管理系统 server：各种接口，以及相应的功能实现；

本人职责： 负责 前台博物馆网站 webgl，资源页模块，后台管理系统 web，后台管理系统 server 全部功能开发。

技术参数：

  1. 前台全景网站 web：js，threejs，createjs，webpack 等；
  2. 资源页模块：js，createjs 等；
  3. 后台管理系统 web：html, css, js；
  4. 后台管理系统 server：express（nodejs）；

软件工具：WebStorm，3dsmax，Blender，Adobe Animate CC
