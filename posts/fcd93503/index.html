1. 在当前项目的目录下，命令行 输入命令

​

​ 备注： ./config-multi/start.js 这段内容是你要调试的webpack配置文件，可以根据实际更改。

​ 命令行里
​ node –inspect-brk ./node_modules/webpack/bin/webpack.config.js –inline –progress

​ node –inspect-brk ./config-multi/webpack.config.js –inline –progress

​ node –inspect-brk ./config-multi/start.js –inline –progress

​ package.json里
​ node –inspect-brk=5858 ./config-multi/start.js

2. 打开chrome，输入 chrome://inspect/#devices
