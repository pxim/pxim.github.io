结构展示 WebGL

config.json 配置表

root

参数 说明 类型 默认值 备注
isAidSystemEnabled 是否开启辅助系统 boolean false
title 网页标题 string “”
sceneBg 背景图路径 string “”
orbitControl 是否开启相机旋转功能 object {“enableRotate”: true} 暂时弃用
angleLimit 相机操作角度限制：水平旋转上限，水平旋转下限，垂直旋转上限，垂直旋转下限，最大向外移动焦距，最大向内移动焦距 object {“maxX”:null, “minX”:null, “maxY”:null, “minY”:null, “maxZ”:1000, “minZ”:120}
nextPage 切换下一个页面的路径 string “”
camera 初始化时，调用模型里的相机，相关设置 object
model 模型相关配置 object
treeData 结构列表树相关配置 object
model
参数 说明 类型 默认值 备注
path 模型的文件夹路径 string “” 参考模板
url 模型的路径 string “” 参考模板
name 模型的名称 string “” 参考模板
parentObj 模型父级容器的相关配置 object {“modelName”: “”, “objectName”: “”} 保持默认设置
visible 模型是否显示 boolean true 保持默认设置
position 模型的位置x,y,z object 保持默认设置
rotation 模型的旋转度x,y,z object 保持默认设置
scale 模型的缩放度x,y,z object 保持默认设置
map 模型的贴图配置 object 保持默认设置，只替换相关图片
treeData
参数 说明 类型 默认值 备注
id 索引 string 参考模板，不能重复
title 菜单的显示文本 string 参考模板，不能重复
key 对应模型里mesh的名称 string 参考模板，不能重复
children 子对象 array 参考模板
