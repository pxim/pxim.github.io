移动端课件包

浏览器支持

PC：Chrome或者Chromium内核 40版本以上，不支持ie

Mobile：微信，企业微信，QQ，Chrome或者Chromium内核浏览器

config.json 配置表

app

网页配置

参数 说明 类型 默认值 备注
title 网页标题 string “”
background 网页背景 string “#9f9f9f” 参考值：#ffffff url(bgimage.gif) api
stage 舞台大小 object {“width”: 1334, “height”: 622 }

enter

开始页配置

参数 说明 类型 默认值 备注
pageId 页面id string “” 参考模板
href 页面路径 string “” 参考模板
format 页面格式（html，video，swf） string “html” 参考模板
type 页面类型 number 1 参考模板
x 页面x坐标 number 0 参考模板
y 页面y坐标 number 0 参考模板
width 页面宽度 number 1334 参考模板
height 页面高度 number 622 参考模板

menu

菜单页配置
具体配置参考 enter

res

资源页列表配置

参数 说明 类型 默认值 备注
mItemId 页面对应的菜单id string “” 参考模板
page 资源页相关配置 string “” 参考模板
page

资源页配置

参数 说明 类型 默认值 备注
pageId 页面id string “” 参考模板
href 页面路径 string “” 参考模板
format 页面格式（html，video，swf） string “html” 参考模板
type 页面类型 number 2 参考模板
x 页面x坐标 number 40 参考模板
y 页面y坐标 number 84 参考模板
width 页面宽度 number 1254 参考模板
height 页面高度 number 520 参考模板
bTitle 页面对应的章节大标题 string “” 参考模板
lTitle 页面对应的章节小标题 string “” 参考模板
